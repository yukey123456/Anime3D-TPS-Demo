Roadmap

Masking

Allows masking so that the amount of deforming by a Lattice can be restrained. Will be achieved through sampling of an existing vertex channel, or a texture.

Lattice Editor Improvements

Add a few requested quality of life features:

  • Soft selection
  • Fit to Renderer/Transform

Expose More Options

Expose more internal settings, to give you more control over desired effects and optimisation:

  • Interpolation type

Utility

Add a few requested utility functions/components:

  • Export/save deformed mesh
  • C# evaluation API for lattices
  • Transform Lattice Modifier (applies Lattices to Transform components)
